#+title: spacemacs快捷键

** 配置相关

| 快捷键      | 说明                  |
|-------------+-----------------------|
| SPC f e d   | 快速打开.spacemacs    |
| SPC f e R   | 同步配置文件          |
|-------------+-----------------------|
| SPC f f     | 打开文件夹            |
| SPC p f     | 搜索文件名            |
| SPC s a p   | 搜索文件内容          |
|-------------+-----------------------|
| SPC f t     | 打开/关闭侧边栏       |
| SPC 0       | 跳转到侧边栏          |
| SPC n(数字) | 光标跳转到n个buffer中 |
| SPC w s     | 水平分割窗口          |
| SPC w v     | 垂直分割窗口          |
| SPC w c     | 关闭当前窗口          |
|-------------+-----------------------|
| ...         | ...                   |
